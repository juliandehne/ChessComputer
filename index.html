ChessComputer by uzuzjmd

Ein eigener Schachcomputer für den Unterricht

In diesem Projekt geht es darum, einen Schachcomputer in der Schule zu entwickeln. Es soll eine einfache Java-Schnittstelle geben, die die Schüler über Scala ansprechen können. Diese soll gut dokumentiert sein. Je nach Fokus des Unterrichts sollen Teile des Schachcomputers, z.B. das Regelwerk für die Züge bereits implementiert sein. Eine einfache KI sollte als Musterlösung implementiert und gut dokumentiert sein. Diese soll folgenden Algorithmus enthalten: 1.) setze Matt 2.) wenn nicht 1: schlage möglichst hochwertige Figur 3.) wenn nicht 2: finde Zug, der die Anzahl der möglichen Züge maximiert

Install

Installation in Netbeans: 1.Netbeans plugin http://plugins.netbeans.org/plugin/36598/nbscala-2-8-x 2.Scala binaries 2.8!! http://www.scala-lang.org/downloads/distrib/files/scala-2.8.2.final.zip 3.Scala zu Path hinzufügen: Rechtsclick auf Computer -> Erweiterte Systemeinstellungen -> Umgebungsvariablen -> Path bearbeiten -> am Ende Pfad zu Scala anhängen http://geekswithblogs.net/renso/archive/2009/10/21/how-to-set-the-windows-path-in-windows-7.aspx 4.Projekt öffnen (Button oben links) 5.Projekt Main Klasse hinzufügen: Rechtsclick auf die Überschrift des Projektes -> run -> bei Mainklasse chesscomputer.scala.Main hinzufügen Dokumentation: Javadoc findet man hier: ChessComputerScala/ownbuild/dokumentation/index.html Scaladoc findet man hier: ChessComputerScala/dist/scaladoc/index.html Das existierende Programm kann man so starten: java -jar "ChessComputerScala\dist\ChessComputerScala.jar"
